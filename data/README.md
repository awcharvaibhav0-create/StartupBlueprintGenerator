# Knowledge Base — Data Directory

Place your startup knowledge base documents here. Supported formats:

- **PDF** (`.pdf`) — will be parsed with PyPDF2
- **Plain text** (`.txt`) — loaded directly

## Suggested documents

| File | Content |
|---|---|
| `startup_india_guide.pdf` | Startup India programme overview and benefits |
| `dpiit_recognition.txt` | DPIIT recognition eligibility and process |
| `msme_schemes.pdf` | MSME registration and government schemes |
| `funding_guide.txt` | Startup funding stages and investor types |
| `business_model_canvas.txt` | BMC methodology and examples |
| `incubators_accelerators.pdf` | Top Indian incubators and accelerators |
| `government_grants.txt` | AIM, MUDRA, PLI, Stand-Up India, CGTMSE |
| `investor_pitch_guide.txt` | Pitch deck structure and tips |

## Built-in knowledge

The application ships with **15 built-in knowledge chunks** covering:

- Startup India & DPIIT recognition criteria
- MSME classification and CGTMSE scheme
- Funding stages (Pre-Seed → Series C) with Indian context
- Angel networks and VC firms active in India
- Business Model Canvas (9 blocks)
- Common revenue models (SaaS, marketplace, freemium)
- Top incubators and accelerators (IIM CIIE, T-Hub, AIM, etc.)
- Government schemes (Atal Innovation Mission, MUDRA, PLI, Stand-Up India)
- Market sizing methodology (TAM / SAM / SOM)
- Pitch deck structure
- Unit economics (CAC, LTV, Payback Period)
- SWOT analysis framework

Any PDF or TXT files you add here will augment this knowledge base automatically on the next app start.
